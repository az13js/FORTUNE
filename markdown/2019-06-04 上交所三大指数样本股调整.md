# 上交所三大指数样本股调整-中新网

　　记者看到，在上证50指标股变化中，山东黄金、绿地控股、大秦铁路、北京银行和三六零等5家公司被踢出。而三一重工、海通证券、中信建投、中国国航以及中国人保等被调入。

　　在上证180指数样本股中，被踢出的有中粮糖业、厦门钨业、中原证券等。而兖州煤业、北汽蓝谷、长沙银行等被纳入。

# 基金业：2020年“买方时代”真正来临-中新网

　　在公募基金产品“下沉”的同时，基金的购买渠道愈发线上化，有第三方理财平台对银行销售渠道进行分流，也有基金公司自家官网购买渠道。此外，伴随投资顾问业务试点的正式落地，基金销售正式向投资顾问转型，基金行业即将进入买方时代。

　　B04-B05版采写

　　新京报记者 张姝欣

　　年化收益

　　2018年末公募基金个人投资者有效账户数达6亿

　　根据中国基金业协会发布的数据显示，截至今年8月，公募基金总规模近14万亿元，我国境内共有基金管理公司126家，其中，中外合资公司44家，内资公司82家；取得公募基金管理资格的证券公司或证券公司资产管理子公司共13家、保险资产管理公司2家。

　　作为目前投资门槛最低的理财工具之一，公募基金自带普惠金融的“基因”，是大众享受金融服务的主要途径。证监会公布的数据显示，截至2018年末，公募基金个人投资者有效账户数已达6亿。

　　其中，货币基金操作便利、零申购费和赎回费、起投金额普遍较低，小额、分散、覆盖面广的普惠金融属性更加明显。以天弘余额宝为例，从2018年报披露的数据看，个人投资者持有份额占比由2017年底的99.94%升至99.97%，随着三四线城市的居民逐渐树立了理财观念和理财意识，天弘余额宝的客户已经遍及全国多个省份，其中有1亿以上的客户来自农村。

　　公募基金发展21年来也为投资者创造了可观的回报。中基协副会长钟蓉萨曾公开表示，中国公募基金的业绩在全球资产管理行业中是很好的，前20年的年化收益超过16%，2019年至今的年化收益超过13%。

　　同时，公募基金行业也积极参与到公益慈善领域。统计数据显示，2010年~2017年7年时间里，公募基金行业为扶贫公益事业捐助4.44亿元，惠及约200万人。

　　销售渠道多元

　　第三方平台上代销基金的规模快速扩大

　　随着移动互联网的发展，基金购买渠道线上化愈发成为明显趋势。

　　第三方平台上代销基金的规模快速扩大，包括天天基金网、支付宝蚂蚁财富等平台，基金产品也更加便利地触及用户，让普通人的理财更容易。根据万德数据显示，截至今年11月2日，按照机构代销的基金数量从高到低排列，前4名均为独立的线上基金销售机构，分别是天天基金网、好买基金、蚂蚁财富和盈米基金。

　　从去年起，BATJ(百度、阿里巴巴、腾讯、京东)已全部杀入基金代销市场。2018年8月，百度拿到基金代销牌照，是BATJ中最后一个获此牌照的公司。在此之前，百度已拥有基金支付牌照，以及基金电商许可。

　　而互联网第三方渠道中，巨头正在后来居上。2017年6月诞生的蚂蚁基金，目前已经接入了近5000只公募基金，超过3500只基金在售。从蚂蚁基金股东恒生电子近三年的年报看，可以发现蚂蚁基金的营收在2018年超越天天基金网。数据显示，蚂蚁基金近三年的营收分别为2.7亿元、7.46亿元、14.04亿元。

　　与此同时，基金公司也对基金进行线上化布局。作为直销方，目前多家基金公司的官网都可以购买基金，且没有渠道费，申购费较低，缺点是注册后只能购买单一基金公司的产品，操作较繁琐。

　　兴全基金首席策略分析师包卫军表示，像支付宝、天天基金对银行销售渠道的分流情况的确存在。随着基金的发展，基民经验也更加丰富，通过银行首次购买基金后，他们也会对特定的一些基金公司，甚至产品有偏好。而互联网平台的申购费用往往更低，这样一来，不少成熟的基民倾向于在第三方互联网平台直接购买单只基金。

　　今年6月末，公募基金管理规模再创新高，已增至13.46万亿元。而与此同时，作为基金销售主力选手的银行，代销基金规模却意外“回落”。记者翻阅A股上市银行中报看到，三大银行代销基金规模或相应手续费增长都出现下滑。例如，其中工行2019年上半年代理销售基金比去年同期代销规模减少了22%；农行的规模较去年同期下降58%。

　　“在渠道费用，也就是申购费上，银行相较于互联网平台，的确比较高。”包卫军说。在他看来，银行是比较强势的渠道。

　　进入“买方时代”

　　公募基金投资顾问业务试点正式落地

　　上个月末，公募基金投资顾问业务试点正式落地，业内评价，基金行业即将进入买方时代。

　　2019年10月25日，证监会发布《关于做好公开募集证券投资基金投资顾问业务试点工作的通知》，首批试点公司包括，华夏基金全资子公司上海华夏财富投资管理有限公司、嘉实基金全资子公司嘉实财富管理有限公司、中欧基金旗下销售公司中欧钱滚滚基金销售有限公司、南方基金和易方达基金5家基金公司。

　　此前，基于“卖方代理”的基金销售模式，客户盈利不高。同时，随着公募基金21年的发展，投资者在众多产品中挑选最合适的产品的难度逐渐加大。

　　基金投顾采用“买方代理”模式，拥有相关资质的投资顾问机构，接受客户委托，在客户授权的范围内，按照协议约定为客户做出投资基金的具体品种、数量和买卖时机的选择，并代替客户进行基金产品申购、赎回、转换等交易申请。也就是说，基金公司实现了与客户利益一致。

　　国创展博宏观研究部首席分析师李瑞华曾公开表示，通过收取一定账户管理费的方式，将基金投顾与客户投资利益更好地进行绑定。这将改变长久以来以“卖方代理”模式为主导的市场格局，也意味着公募基金“买方时代”的真正到来。

　　展望

　　海外资金有望加速流入

　　去年6月1日A股被纳入MSCI指数，到了今年11月MSCI已实现年内第三次扩容。这意味着，A股投资者结构将被重塑。国金策略发布的数据显示，当前机构投资者中，从持有流通市值占比角度来看，公募占比最高为4.52%。

　　跟踪相关指数的MSCI主题基金也迅速申报和成立，在市场上火起来。根据wind数据显示，截至2019年11月2日，MSCI相关基金共计40只，其中招商基金发行的MSCI中国A股国际通指数型基金规模最大，达49.17亿。

　　长量基金资深研究员王骅也公开表示，随着未来MSCI继续扩大中国A股纳入系数，海外资金也有望持续加速流入，或推动成分股乃至整体市场走牛，对应的MSCI主题基金也具有长期配置的价值。

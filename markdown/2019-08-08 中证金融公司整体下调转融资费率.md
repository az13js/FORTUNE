# 中证金融公司整体下调转融资费率-中新网

　　该公司信息显示，中国证券金融股份有限公司是证券公司开展融资融券业务重要的资金来源渠道之一，通过证券公司融资业务对资本市场提供流动性支持，满足投资者融资需求。此次下调转融资费率，是根据资金市场利率水平做出的调整，有利于降低证券公司融资成本，促进合规资金参与市场投资，维护我国资本市场平稳健康发展。

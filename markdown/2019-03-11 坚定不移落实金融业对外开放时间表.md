# 坚定不移落实金融业对外开放时间表-中新网

　　本报讯（记者 王卓）3月10日上午，十三届全国人大二次会议举行记者会，中国人民银行行长易纲，副行长陈雨露，副行长、国家外汇管理局局长潘功胜，副行长范一飞就“金融改革与发展”相关问题回答记者提问。

　　“稳健的货币政策是一个内容非常丰富的政策取向。”易纲说，稳健的货币政策要体现逆周期的调节，同时货币政策在总量上要松紧适度。今年的松紧适度，就是要把广义货币（M2）和社会融资规模的增速大体上和GDP名义增速保持一致。

　　除此之外，稳健的货币政策还要兼顾内外平衡。易纲表示，中国的经济已经深度融入了世界经济，考虑货币政策的时候，要同时兼顾中国外向型经济的特点和在全球经济关系中的地位。

　　“去年习近平主席在博鳌亚洲论坛发表主旨演讲，提出一系列对外开放重大举措，我们将尽快使之落地。”易纲说，“其中一些有了重要的进展，比如标普公司已经获得进入中国信用评级市场的准入，美国运通的筹备申请已获得批准，等等。我们相信，中国金融市场的开放对中国有利、对世界有利，所以我们会坚定不移地按照金融业对外开放时间表来推进。”

　　陈雨露介绍说，目前的征信体系是“政府+市场”双轮驱动的发展模式。其中，人民银行征信中心负责的国家信用信息基础数据库，已接入3500多家银行和其他金融机构的信用信息数据，9.9亿自然人的信用信息，2600多万户企业和其他法人组织的信用信息。

　　“目前在市场上，我们已经有125家企业征信机构，还有97家信用评级机构，这些机构80%以上都是民营资本投资兴办的。”陈雨露说，“人民银行批准了我国首个市场化的个人征信机构——百行征信有限公司。经过一年多的筹备，目前百行征信有限公司已经签约接入了600多家机构的信用信息。今年1月份已经正式推出了个人信用报告等3项征信服务产品，实现了良好的开局。”

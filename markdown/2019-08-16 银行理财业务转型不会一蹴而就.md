# 银行理财业务转型不会一蹴而就-中新网

　　银行理财业务过去通过资金池的运作模式享受了不少“红利”，未来净值型产品转型也面临新的机会与困难。权衡下来，理财业务转型并不会一蹴而就。刚设立和正在筹建的理财子公司会沿着转型成本小、潜在红利多的方向推进。整体来看，预计产品配置上对非标资产依赖少、标准化风险资产尤其是权益资产配置比例高，负债成本高、注册地比母行更为市场化、管理层与核心业务层来源于非母行比例高的理财子公司后续在推进净值化转型进度上会更快。

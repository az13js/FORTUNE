# 新生儿要买哪些保险？-中新网

　　理财问答：

　　文/广州日报全媒体记者陈明

　　问：作为普通工薪家庭，家有新生婴儿，需要为宝宝购买什么样的险种？

　　答：新生婴幼儿是保险公司重点关注的客户群体之一，因为保险公司如果获得了一位低年龄客户的认同，双方合作愉快的话，那么伴随这位客户的成长，将获得一个长期而可靠的合作伙伴。由此，婴幼儿的保险相对来说价格较之成年人要更优惠。

　　婴儿需要购买的险种应该是人身保险为主，理财保险为辅，其中尤其是大小病医疗险、重大疾病保险、意外险等。对于工薪阶层来说，这是避免和防范家庭成员因病致贫、因病欠债的最好途径之一。

　　问：不少针对婴幼儿的险种都是将未来的教育费用、医疗费用结合到一起的，这样的综合险有意义吗？

　　答：经过多年的运行表明，普通家庭购买保险适合针对性强、指向明确的险种，而综合险费用更高，赔付率相对而言更低。普通工薪家庭在购买基本保险时，因为金额普遍不大，占用资金不多，可以不用考虑理财投资类的功能，而只需要认真研究人身保险的功能和赔付率即可。

　　问：意外身故，为何未成年人和成年人获取赔偿金区别很大？

　　答：或许是出于国际惯例，目前的人身保险合同大多数都约定“未成年身故时，保险寿险责任赔付应按照0~9岁最多赔20万元；10~17岁最多赔50万元”来进行赔付，即使是投保人缴纳再多的保费，对于未成年人也是这个定额。而成年人则根据投保费用的多寡来设定身故赔偿金的额度。因此，给婴幼儿投保，重点需要研究重大疾病、意外伤害这一类的医疗费用的赔付问题，如此才能更加科学地制定投保计划。

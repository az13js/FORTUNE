# IMF下调全球经济增长预期 专家呼吁协商解决经贸摩擦-中新网

　　张涛指出，一系列数据显示，全球经济正在经受经贸摩擦的负面冲击。他呼吁有关各方通过协商找到可持续方案，有效解决分歧。

　　

　　

　　国际货币基金组织在7月份发布的一份报告中再次下调全球经济增长预期，分别将2019年和2020年世界经济增速下调至3.2%和3.5%。其中，英国脱欧和经贸摩擦带来的不确定性是国际货币基金组织作出调整的重要因素。

# 央行开展300亿元逆回购操作 -中新网

　　央行发布公告称，为维护半年末流动性平稳，当日开展300亿元逆回购操作，期限为14天，中标利率为2.7%。

　　当日，银行间市场各期限利率涨跌互现。20日发布的上海银行间同业拆放利率(Shibor)显示，隔夜利率为1.253%，较上一个交易日下行19.4个基点；7天和14天利率分别为2.448%、2.859%，较上一个交易日上行6.2个和7.2个基点；一年期利率为3.207%，较上一个交易日下行1.1个基点。

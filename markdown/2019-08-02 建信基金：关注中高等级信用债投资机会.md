# 建信基金：关注中高等级信用债投资机会-中新网

　　近期债券市场呈现出两大特征：一方面资金面宽松，市场整体流动性得到了极大提升；另一方面，新增债券违约主体和违约金额有所增加，进行债券投资时需要更加用心甄别。建信基金认为，叠加国内外多重因素影响，三季度市场整体流动性将大概率维持宽松预期，中高等级信用债投资机会将成为下半年关注重点。

　　据了解，建信基金已搭建起一只专业投研团队，并始终将风控作为公司健康发展的基石，持续提升风险管理效率和水平。
